My Work

A collection of projects I've worked on.

Banking Data Pipeline on Azure

Built a full Medallion Architecture on Azure simulating a real multi-bank environment. Raw transactions from two banking systems, flat files, and an FX API flow through Bronze, Silver, and Gold layers — orchestrated by ADF, transformed in Databricks, and governed with audit logging and SCD2 modelling.

Analytics Engineering with dbt

A structured analytics engineering project applying dbt best practices on an ecommerce dataset. Raw CSV data flows through staging, intermediate, and mart layers into clean dim_customers and fct_orders tables — with built-in data quality tests, reusable macros, and automated validation on every push.

Financial Data Dashboard

A data analytics project combining Python and Tableau to examine Luxembourg's economic structure over a decade. Covers GDP growth, sectoral volatility, risk–growth trade-offs, and tax ratio dynamics — built on a reproducible Python pipeline using official LUSTAT and Eurostat datasets.
